# Cases (Sektion 6) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a new `Cases.astro` section (Sektion 6, "Was daraus entsteht.") showing 4 cases in a Dreiklang (Ausgangslage / Was wir taten / Jetzt) with a scroll-animated process-arrow diagram, and wire it into `index.astro` as the new Sektion 6, renumbering subsequent section comments.

**Architecture:** New `src/components/sections/Cases.astro` renders an editorial list of 4 case blocks (mono label, process-arrow SVG with 1 line + 3 nodes, 3-column Dreiklang text, "Zur Case Study" link), reusing the existing `Section.astro` wrapper, `initScrollAnimate`, and `initConnectionDiagram` (no new animation code needed). `index.astro` gets a new import and a new `<Cases />` placement directly after `<Leistungen />`, with the section comments for the 5 sections that follow renumbered from 6–10 to 7–11.

**Tech Stack:** Astro 5, TypeScript strict, Tailwind CSS 4 (CSS custom property tokens from `src/styles/global.css`), existing `src/lib/scrollAnimate.ts` helpers (`initScrollAnimate`, `initConnectionDiagram`).
